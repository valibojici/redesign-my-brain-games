Use the arrow keys to choose if the object shown is a tool (LEFT ARROW) or writing utensil (RIGHT ARROW). For every 6 correct answers the image duration decreases and for every 3 wrong answers the image duration increases. The default image duration is 300ms. The default object count is 50.
